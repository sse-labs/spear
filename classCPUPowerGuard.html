CPUPowerGuard Class Reference

#include <CPUProfiler.h>

Detailed Description

RAII Guard to ensure that the CPU performance mode is enabled during profiling and disabled afterwards, even if an error occurs. The constructor enables the performance mode and the destructor disables it, ensuring that the CPU settings are properly restored after profiling is done.
